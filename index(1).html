🛠️ 长征数字地图 · 完整技术方案
TECHNICAL ARCHITECTURE & DATA STRUCTURE SPECIFICATION
🗺️
Layer 1 · 底图层 (Base Map)
推荐方案：Leaflet.js + OpenStreetMap / 天地图

技术选型理由：
• Leaflet 轻量级（42KB gzipped），适合教育场景
• 支持多底图切换：标准地图 / 卫星影像 / 历史地图
• 高德地图API可提供中文地名标注
• Mapbox GL JS 可实现3D地形效果

数据格式： TileLayer WMS XYZ Tiles
// 底图配置示例 { "baseMap": { "type": "TileLayer", "url": "天地图API", "attribution": "©天地图", "maxZoom": 18, "minZoom": 4 } }
📍
Layer 2 · 线路层 (Routes Layer)
数据格式：GeoJSON LineString

三大方面军路线：
• 红一方面军：江西瑞金→陕西吴起（红色 #e74c3c）
• 红二方面军：湖南桑植→甘肃会宁（蓝色 #3498db）
• 红四方面军：四川→甘肃（绿色 #2ecc71）

路线属性： army_id start_date end_date distance_km
{ "type": "FeatureCollection", "features": [{ "type": "Feature", "geometry": { "type": "LineString", "coordinates": [ [116.03, 25.88],// 瑞金 [106.93, 27.69] // 遵义 ] }, "properties": { "army": "第一方面军", "color": "#e74c3c", "weight": 3 } }] }
📅
Layer 3 · 事件层 (Events Layer)
数据格式：GeoJSON Point + MongoDB / JSON

事件分类：
• 🔴 重要战役 (battle)
• 🟡 重要会议 (conference)
• 🔵 重要渡口/关隘 (geography)
• 🟣 重要会师 (reunion)

字段结构： event_id title date location description images[] significance
{ "event_id": "EVT_001", "title": "遵义会议", "date": "1935-01-15", "coordinates": [106.93, 27.69], "type": "conference", "army": 1, "images": ["url1", "url2"], "significance": "生死攸关的转折点", "tags": ["转折点", "毛泽东"] }
🏔️
Layer 4 · 地理层 (Geography Layer)
数据格式：GeoTIFF / DEM / Terrain RGB

地理要素：
• 山脉：夹金山、岷山、六盘山
• 河流：湘江、乌江、赤水河、金沙江、大渡河
• 高原：川西高原、若尔盖草地
• 关隘：娄山关、腊子口

技术实现： DEM数据 等高线 Mapbox Terrain 3D地形
{ "geography_id": "GEO_001", "name": "夹金山", "type": "mountain", "elevation": 4114, "crossed_date": "1935-06-12", "difficulty": "极高", "casualties": "冻伤无数", "polygon": {"type": "Polygon"} }
👤
Layer 5 · 人物层 (People Layer)
数据格式：GeoJSON FeatureCollection (时序点集)

主要人物：
• 毛泽东 / 周恩来 / 朱德 / 彭德怀
• 林彪 / 刘少奇 / 邓小平 / 陈云

轨迹字段： person_id trajectory[] role key_events[]
{ "person_id": "P001", "name": "毛泽东", "role": "中央政治局委员", "trajectory": [ { "date": "1934-10-16", "location": [116.03,25.88], "event": "长征开始" } ] }
⚔️
Layer 6 · 战役层 (Battles Layer)
数据格式：JSON + SVG Animation

主要战役：
• 湘江战役 (1934.11) - 最惨烈
• 四渡赤水 (1935.1-3) - 最精彩
• 飞夺泸定桥 (1935.5.29) - 最传奇
• 直罗镇战役 (1935.11)

战役字段： battle_id phases[] forces outcome significance
{ "battle_id": "B002", "name": "四渡赤水", "period": "1935-01至03", "phases": [ {"phase":1, "date":"1935-01-29"}, {"phase":2, "date":"1935-02-18"} ], "red_army": 30000, "enemy": 400000, "outcome": "红军成功突围" }
